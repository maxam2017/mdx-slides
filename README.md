## Overview
Markdown-driven slideshows built with modern frontend libraries
## Dependency
- Next.js
- tailwind.css
- react-spring
- react-use-gesture
- mdx
- prism-react-renderer

## Setup
```bash
yarn install # install dependencies
yarn prepare # for husky git hook
yarn dev # run next.js project
```
